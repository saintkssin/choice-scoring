# Choice AI Call Scoring

Автоматична система оцінки якості дзвінків сейлсів по скрипту продажів.

## Чому Gemini, а не ChatGPT + Whisper

Після аналізу вартості підписок було прийнято рішення не використовувати набір інструментів ChatGPT + Whisper.

**Вартість стека з ТЗ:**
| Сервіс | Вартість |
|--------|----------|
| OpenAI API ключ | $5 |
| ChatGPT Pro (160+ запитів/добу) | $120/міс |
| Whisper (2000+ хв транскрипції) | $25/міс |
| **Разом** | **~$150/міс** |

**Для порівняння — Ringostat має вбудовану транскрипцію:**
- $0.01–0.03 за хвилину залежно від мови
- Для України: $0.03/хв
- 2000 хвилин = $60 — вже дешевше

**Наш стек (Gemini API):**
| Сервіс | Вартість |
|--------|----------|
| Транскрипція аудіо | ~$0.30 за 15 дзвінків |
| Скоринг через Gemini | ~$0.10 за 15 дзвінків |
| **Разом** | **~$0.40 за запуск** |

Gemini 2.5 Flash підтримує аудіо нативно — транскрипція і скоринг в одному API без додаткових підписок.

---

## Як це працює
1. `1_fetch_calls.py` — завантажує вихідні дзвінки з Ringostat за останні 14 днів з фільтром по менеджерах
2. `2_transcribe.py` — транскрибує аудіо через Gemini 2.5 Flash
3. `3_score.py` — оцінює транскрипт по 6 блоках скрипту через Gemini 2.5 Flash
4. `4_report.py` — збирає результати у CSV та детальний TXT звіт

---

## Запуск

### Одною командою (повний pipeline)
```bash
pip install requests google-genai
python3 run.py
```

### Або покроково
```bash
python3 1_fetch_calls.py
python3 2_transcribe.py
python3 3_score.py
python3 4_report.py
```

## Конфігурація (`config.py`)

| Параметр | Опис |
|----------|------|
| `RINGOSTAT_API_KEY` | API ключ від Ringostat |
| `GEMINI_API_KEY` | API ключ від Google AI Studio |
| `MIN_DURATION` | Мінімальна тривалість дзвінка в секундах (за замовчуванням 60) |
| `MANAGERS` | Список менеджерів для фільтрації по полю `caller`. Вказати прізвища сейлсів (наприклад `["Sporysh", "bondarchuk"]`). Порожній список = всі вихідні дзвінки |
| Дати | Автоматично — останні 14 днів |

---

## Результати

- `output/report.csv` — таблиця з оцінками по всіх дзвінках
- `output/report_detailed.txt` — детальні коментарі та цитати по кожному дзвінку
- `output/scores/` — JSON файли з оцінками по кожному дзвінку окремо

---

## Блоки скрипту (script.json)

| # | Блок | Що оцінюється |
|---|------|---------------|
| 1 | Аналіз клієнта | Чи згадував аналіз закладу перед дзвінком |
| 2 | Встановлення контакту | Представлення, вихід на відповідального |
| 3 | Виявлення потреби | Питання про доставку, замовлення, агрегатори |
| 4 | Презентація | Вигоди під клієнта, приклади, інтеграції |
| 5 | Робота із запереченнями | Відповідь на "дорого/не зараз/є конкурент" |
| 6 | Закриття | Онлайн-зустріч, конкретний час, контакт |

---

## Обмеження

- Gemini розділяє спікерів як [МЕНЕДЖЕР]/[КЛІЄНТ], але точність розподілу залежить від якості аудіо
- Ringostat API не підтримує фільтр по типу дзвінка через параметр filters — фільтрація вихідних реалізована на стороні коду по полю `caller`
- Ringostat API не повертає ім'я менеджера окремим полем — ідентифікація по імені в полі `caller`

## Що зроблено понад ТЗ

- Запуск одною командою `python3 run.py` — повний pipeline без ручного втручання
- Автоматичні дати — останні 14 днів без ручного налаштування
- Фільтр дзвінків менше 60 секунд — автоматично відкидаються короткі дзвінки (не розмови)
- Фільтр по конкретних менеджерах через `MANAGERS` у config.py
- Повторний запуск пропускає вже оброблені файли
- Автоматична обробка помилок з retry та затримкою
- Вартість рішення $0,35 за 45 дзвінків тривалістю від 60 секунд проти ~$150/міс за стеком з ТЗ
